JavaScript - A linguagem da web

Sites com um visual invejável e cheios de conteúdo? nem sempre foi assim. Neste post vamos mergulhar no mundo de uma linguagem que certamente mudou a forma de interagir com a web, o JavaScript.

O início de tudo

Bem ... nos primórdios da internet, ano de 1995, um programador formado em matemática e ciências da computação pela Universidade de Santa Clara chamado Brendan Eich (também um dos fundadores da Mozilla em 1998) foi contratado pela NetScape Corporation (produtora do primeiro navegador comercial em 1994, o NetScape Navigator) no desenvolvimento de um projeto chamado Mocha (.. e também de LiveScript um tempo depois), cujo objetivo era tornar a navegação na internet mais dinâmica, e então surgiu a primeira “pegada” do JavaScript. Na época a web era estática, se resumindo a conteúdo de texto com poucas imagens estáticas, um ambiente extremamente entediante para os padrões atuais.

Deriva do JAVA?

Na realidade acredita-se que o nome JavaScript foi uma estratégia de marketing, visto que na época a linguagem JAVA tinha sido lançada recente, portanto JAVA e JAVASCRIPT não possuem nenhuma ligação.

De patinho feio a queridinha da web

Por ser uma linguagem desenvolvida para a web, no contexto da década de 90, o javascript não se notabilizou como linguagem "confiável" em seu início de jornada, pois diferentemente de linguagens mais robustas como JAVA, C e C# por exemplo, não possui checagem de tipos (sendo possível uma variável iniciado com números inteiros receber um valor string por exemplo) e de sua característica Single-Thread, fator bloqueante para execução de multiplos processamentos (possui apenas uma fila de execução).

Porém com o surgimento do V8 (engine desenvolvida pelo google para multiprocessamento no javascript), foi permitido ao javascript a execução de tarefas de multiprocessamento Isso ocorre pelo fato de na thread do javascript são executadas tarefas não-bloqueantes, e para tarefas assíncronas e bloqeuantes como "callbacks" e "promises" por exemplo, serem executadas com o apoio de uma web API desenvolvida em C++. Esse foi o ponto de virada na jornada do javascript para se tornar Hoje a linguagem mais utilizada segundo relatórios anuais do Stackoverflow.

Como funciona o JavaScript?

Por ser uma linguagem de script, o javascript é inserido através de uma tag de script em um arquivo HTML, que é representado no navegador através da árvore do DOM(Data Object Model – Um objeto que permite aos navegadores a interpretação de arquivos HTML). Hoje em dia, a maioria dos navegadores possuem integrados os recursos que permitem a interpretação de código javascript.gem JAVA tinha sido lançada recente, portanto JAVA e JAVASCRIPT não possuem nenhuma ligação.

Em que projetos aplico o JavaScript?

O javascript permite o carregamento de recursos da web de forma assíncrona (sem a necessidade de recarregar toda a página, o famoso “F5”), facilitando o tráfego de informações entre sites e sistemas web com um ótimo desempenho e economia de recursos. Esse carregamento assíncrono é realizado através de AJAX (Assynchronous Javascript and XML), onde os dados são transportados em formato JSON (objeto possuindo apenas chave e valor) e protocolo Websocket.
